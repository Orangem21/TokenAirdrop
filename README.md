# ERC20TokenAridrop
This project can be used for all ERC20 based tokens for airdrop distribution！

### 使用说明

1. 部署Airdrop.sol 合约 
2. 转账给合约足够的代币
3. 完善表格  修改Config
4. node awards_airdrop.js
